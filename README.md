# 🎓 ShikshaMitra AI

## 🚀 Overview

ShikshaMitra AI is an AI-powered educational platform designed to make learning smarter, faster, and more interactive.

Students can upload textbooks and syllabi, chat with AI, generate quizzes, create flashcards, and receive personalized study plans.

---

## ✨ Features

- 🤖 AI Tutor
- 📷 OCR Textbook Scanner
- 📄 PDF Learning
- 💬 Chat with PDF
- 📝 AI Quiz Generator
- 🃏 AI Flashcards
- 📅 AI Study Planner (Syllabus PDF Upload)
- 📊 Learning Progress Dashboard

## ⚙️ Installation

### Backend

```bash
cd Backend
pip install -r requirements.txt
python run.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
